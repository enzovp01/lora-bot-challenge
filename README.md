## 🚀 Instalando Lora Bot

Para instalar o Lora Bot, siga estas etapas:

* Clone o repositorio
* Instale as seguintes bibliotecas:
```
npm install express
npm install axios
```
* Acesse o diretório "API"

* Instale as dependências:
```
npm i
```

* Execute o projeto com:
```
npm start
```
